# Unit test

## Test runner

테스트를 실행 후 결과를 생성

ex) mocha, karma

## Assertion

테스트 조건, 비교를 통한 테스트 로직

ex) chai, should.js, expect.js, better-assert

jest를 사용하면 위 두가지를 동시에 가능

# Jest

https://jestjs.io/

- zero config
- snapshots
- isolated
- great apis (it, expect)
  - https://jestjs.io/docs/api
- mocking
- exceptional error messages
